Starlette

Starlette is a modern async python web framework for building high performance apps. It's perhaps best known for the FastAPI framework, which extends Starlette. I first encountered starlette in my role at Black Cow, where I contributed to the development of innovative multiplayer online games.

Since embracing the new async style of Python, I've enjoyed working with Starlette - in fact this app is served by a Starlette + Uvicorn solution.

Flask

Flask was my initial entrypoint into web-development. Flask prides itsself on a very 'hands-off', un-opinionated approach that contrasts sharply with some of it's contemporaries like Django.

I'm grateful that I started with Flask because it gave me the opportunity to mistakes that perhaps another framework wouldn't have. Those mistakes taught me the value of a lot of the web-development design-patterns and best practices that I swear by today.

.NET core

I used C#'s .NET core runtime during my role Zendbox, where I built an API that powered a warehouse management dashboard.
